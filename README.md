# 项目描述
仿商城做的一个系统，电商或电商类型的服务在目前来看依旧是非常常用，虽然纯电商的创业已经不太容易，但是各个公司都有变现的需要，所以在自身应用中嵌入电商功能是非常普遍的做法。
          为了让大家掌握企业开发特点，以及解决问题的能力，我们开发一个电商项目，项目会涉及非常有代表性的功能。
          为了让大家掌握公司协同开发要点，我们使用git管理代码。
          在项目中会使用很多前面的知识，比如架构、维护等等。
